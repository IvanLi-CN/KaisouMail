# 组件预览 / Storybook

Storybook 是这套公开站点的一部分，不是附带产物。

## 用途

- 稳定查看关键页面和组件状态
- 为界面改动提供可重复的视觉证据
- 在不登录控制台的前提下共享 UI 预览

## 发布方式

- 文档首页：`/`
- Storybook 静态站：`/storybook/`
- 跳转页：`/storybook.html`

文档站顶部导航会直接暴露 Storybook 链接，README 也会同步给出入口。

## 推荐查看内容

- Domains 页面
- Mailbox create card
- API Keys / 速查文档入口
- Workspace 三栏布局

## English summary

Storybook is published alongside the docs site on GitHub Pages and acts as the public component review surface for CF Mail.
